GoBD Archiv – Web Interface

Ein Zugang zu dem GoBD-konformen Archivsystem, das durch ein strenges Rollenkonzept und eine lückenlose Historisierung sicherstellt,

Es bietet einen direkten Datenzugriff für Betriebsprüfer, sowie den Export steuerrelevanter Unterlagen.
